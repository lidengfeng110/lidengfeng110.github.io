实验：yum包安装 mysql 数据库

centos 6 上装 yum install mysql-server

来看看他的包结构       rpm -ql mysql-server  
    /etc/rc.d/init.d/mysqld      这是服务脚本      
 mysqld     这是服务名  
/usr/libexec/mysqld       这是数据库的主程序，二进制程序路径

/var/lib/mysql 这个目录就是将来存放用户数据的数据库，这是yum 安装的默认路径，当然将来如果你是二进制安装或编译安装也可以自己选择路径。
/var/log/mysqld.log 这是日志路径
装上包后启动服务并设为开机启动 chkconfig mysqld on service mysqld start
启动后会自动打开 3306 端口
启动服务后会生成数据库文件 /var/lib/mysql/

现在就可以用客户端工具连接数据库了，他的客户端工具就是 mysql ,看看客户端工具属于哪个包的，如下图：


接下来就可以使用客户端工具进行连接了。 这是交互式的。
进去后敲 help 可以查看客户端工具使用的命令。
敲 show database 命令可以查看当前的数据库列表

可以看到只有三个，而每个数据库可以存放很多列表。

在mysql 中也可以执行linux 命令。


也可以查看状态信息

可以进入到某个数据库。

要退出敲 qiut 或 exit 都可以。

centos 7

先装包 yum install mariadb-server
装好包后他会也把客户端工具 mysql 装上
mariadb-service 这是服务名
/uer/libexec/mysqld 这是主程序路径
然后启动服务 systemctl start mariadb
启动完后他应该就打开了 3306 端口
还有服务没启动前， /var/lib/mysql/ 目录是空的

服务以启动 /var/lib/mysql/ 目录会自动生成文件

也可以查看当前有什么数据库

查看某个数据库的列表可以用命令 show tables 来查看
比如显示 user 这张表的内容可以用命令 select from user;
有时候内容过多显示内容的格式不容易看，可以加个选项\G,如：select from user\G
如果删除数据库可以用命令 drop 。
如删除 mysqlclone 这个数据库， 如： drop database myaqlclone;

怎么查看mysql 中的用户信息呢？
先切换进 mysql 数据库 use mysql; 用命令查看数据库的列表 show tables;
打开列表后就有一个 user 这个表，这个列表中就存放着 mysql 中的用户信息。
如果想查看这个表的内容，可以用命令 select * from user\G 来查看
不过这个表中的内容也很多，也可以挑着列来显示
select user,password,host from user;

这表示只挑这三种来显示

mysql 刚安装好是不安全的，要提高安全性需要跑一个脚本，这个脚本是 mysql_secure_installation 这是个系统自带的脚本
直接运行这个脚本，他会提示你输入当前root 的口令，如果当前没口令就直接回车，接下来会提示是不是设下口令，敲 y 。然后敲两遍 口令。 接下来它会显示是否删除匿名账号，敲 y 。接下来显示是否禁用远程登录呢。这个看生产环境决定，这块先敲 n 。接下来删除测试数据库，如果生产中也不测试建议把他删除，敲 y 。接下来会显示是否让以上设定的权限生效，在这敲 y 。到这这个脚本就跑完了。
这样下次直接敲 mysql 就不能登录了，必须输入用户名和口令。如： mysql -uroot -pcentos
连接上 可以用命令 select user(); 可以查看当前mysql 用户
在mysql 中的用户名跟linux 的不一样，这里的用户名是 用户名加主机名组成的。
user=username@host 中的host 是指从哪台主机登录。 如： root@locahost
在 mysql 中默认是不支持远程连接的，如果在远程主机上连接本机数据库，格式是：
如： mysql -uroot -pcentos -h192.168.30.7
但是要想连接要给远程主机授权才能连接。
